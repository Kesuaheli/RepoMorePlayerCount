# MorePlayerCount

A Repo mod that allows to have more player count upgrades for the map.

Ever wanted to have more than only one player have a 'Map Player Count' upgrade?

### What it does...

The mod takes the amount of players currently in the game and use that to determine the maximum amount of upgrades able to buy.

Up to **half the number of players in your lobby** of upgrades can be bought.

### Examples

| Players in your lobby | Maximum number of Map Player Count upgrades |
|-----------------------|---------------------------------------------|
| 1 (or Singleplayer)   | 0 [^1]                                      |
| 2-3                   | 1 (unchanged)                               |
| 4-5                   | 2                                           |
| 6-7                   | 3                                           |
| 8-9                   | 4                                           |
| ...                   | ...                                         |

[^1]: A special case when you're playing alone: No Map Player Count upgrade will be able in the shop. Well, most of the time you know if you are still alive anyway... hopefully.
